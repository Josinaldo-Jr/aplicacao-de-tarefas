# task-manager-react
Aplicação React para gerenciamento de tarefas (To-Do List), desenvolvida com React Router, Context API e Hooks. O sistema permite adicionar, visualizar, editar e remover tarefas, com persistência de dados via localStorage e navegação entre páginas.

---

## Objetivo

Praticar conceitos intermediários e avançados do React, incluindo gerenciamento de estado global com Context API, utilização de Hooks, persistência de dados e navegação entre páginas utilizando React Router.

---

## Funcionalidades

- Cadastro de novas tarefas
- Visualização da lista de tarefas
- Edição de tarefas existentes
- Remoção de tarefas
- Navegação entre páginas com React Router
- Gerenciamento global de estado com Context API
- Persistência de dados com localStorage
- Manipulação de eventos e renderização reativa
- Estrutura organizada e componentizada
